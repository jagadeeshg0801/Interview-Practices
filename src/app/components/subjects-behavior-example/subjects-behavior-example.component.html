Note: An observer is a data producer of multiple values, "pushing" the values to Observers (Consumers). We can consume the data produced by the observables using the .subscribe() method of an observable. Observable has three callback functions
  • next() :- calls when observable changes value.
  • complete() :- calls when observable completes.
  • error() :- calls when observable throws an error.
