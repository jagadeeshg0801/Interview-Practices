Note: Interceptor Usage is, For suppose we need to add some Request headers in API to send it to backend, then we use the interceptors. here we added API_KEY in the headers by using Interceptor

  • Step 1: Create a Service and implements HttpInterceptor
  • Step 2: In the same service implement intercept() method and override with your desired logic (like adding headers to the API)
  • Step 3: Import this Service in Provider's section of app.module.ts , config like this provide: HTTP_INTERCEPTORS , useClass: YourClassName
  • After loading this screen , just check API Request headers in the browser's tab, then you can see the added api_key like shown in the image
